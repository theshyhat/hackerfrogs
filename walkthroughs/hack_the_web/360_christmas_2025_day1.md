# URL
https://hack.arrrg.de/challenge/360
# Concept
* disabling JavaScript in the web browser
# Method of solve
* there was a JavaScript function on the webpage which causes snow graphics to cover the screen and the text
* we need to disable JavaScript to read the webpage and get the answer
* in the browser, we can open up the Developer Tools, then go to settings from the triple dot menu button, and select `Settings`
* in the `Advanced Settings`, we see a checkbox labeled `Disable JavaScript`
* check that box, and the snowflakes disappear and we read the answer
