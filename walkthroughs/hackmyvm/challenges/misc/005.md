# URL
https://hackmyvm.eu/challenges/challenge.php?c=005
# Concept
* cracking Linux password hashes
# Method of solve
* this challenge wants us to crack Linux password files, and we're given the contents of a `shadow` file
* we need to unshadow the file along with a `passwd` file
* the user we're targetting is `root`
* we can create a `passwd` file with the following contents:
```
root:x:0:0:root:/root:/bin/bash
```
* and then pull out the line with the root user's shadow entry and save it to `shadow`
* next, we unshadow the file with this command:
```
unshadow passwd shadow > unshadow.hash
```
* finally, we use `john` to crack the hash:
```
john --wordlist=/usr/share/wordlists/rockyou.txt unshadow.hash
```
* done done done
