# URL
https://hackmyvm.eu/challenges/challenge.php?c=008
# Concept
* cracking zip password hashes
# Method of solve
* we need to unzip the `flag.txt` file, but the zip file is password protected
* to extract the password hash, we use `zip2john`:
```
zip2john 008.zip > zip.hash
```
* then we can use `john` to crack the hash:
```
john --wordlist=/usr/share/wordlists/rockyou.txt zip.hash
```
* lastly, we need to use the right program to access the AES-encrypted zip file:
```
7z x 008.zip
```
* done
