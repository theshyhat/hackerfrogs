# Concept
* keyboard-shift cipher
# Method of solve
* this ciphertext was encrypted using the keyboard-shift cipher, which is a cipher where each letter in the plaintext is substituted with the key in a specific direction on the keyboard, one key left, or right, for example
* this keyboard-shift cipher is using the standard qwerty keyboard, and the shift is the letter to the right of the plaintext letter
