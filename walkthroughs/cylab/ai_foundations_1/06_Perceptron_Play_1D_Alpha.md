# Concept
* playing with perceptron weights to fit classification model
# Method of solve
* this challenge wants us to adjust a perceptron's bias and weight values until it can classify some values into the correct categories
* we can play around with the weights and bias values, but this pair should get the job done:
```
weight=2, bias= -2
```
