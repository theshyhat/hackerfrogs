## Enumerate Bucket Contents without Credentials
```
aws s3 ls <bucket_name> --no-sign-request
```
## Enumerate Bucket ACL (Access Control List)
```
aws s3api get-bucket-acl --bucket <bucket_name>
```

## Enumerate Bucket Policies
```
aws s3api get-bucket-policy --bucket <bucket_name>
```
## Download Bucket Objects
```
aws s3 cp s3://your-bucket-name/file.txt .
```
* sometimes we need to add `--no-sign-request` to our command if we don't have credentials
