# Overview
This is a tool for brute-forcing credentials on the AWS web console. Note that this tool does password spraying / credential stuffing
# Installation (Kali)
```
GoAWSConsoleSpray
```
# Use
The tool should be installed in the `~/go/bin` directory
```
./GoAWSConsoleSpray -a 243687662613 -u users.txt -p pass.txt
```
